Informative

This Informative Page is our first section, where it is briefly discussed what problems can appear these days when we want to send a file, and with what solution we came up to solve these problems.

The text is followed by a clickable button, which directs you to the login/register section.

Login and Register

In order to access our application features, a person has either to log in with his account, either to register with an valid and non-existing username , email address and password.

If all credentials are validated, then the user is redirected to the home page.

  • In order to Log In or Register, all fields below are required!

Navigation Bar

This is our Navigation Bar, that allows a user to navigate through our application pages.

On the left side there is our STOL logo.

On the right side we have 3 buttons:

  • "Home" - redirects an user to home page, where he can see various information about his storage space or his uploaded files
  • "My account" - redirects an user to a settings page, where he can see to which cloud storage accounts is connected to, and to which he can connect to.
  • "Log Out" - disconnects a user from his account, redirecting him to the main informational page

Home Page

On application's home page, we can see various information for our account, such as:

  • A welcome message with your username, so you can see that your account is successfully logged
  • Available remaining space for uploading data
  • A progress bar that shows us the used space, and also this progress bar automatically updates his status when a file is deleted or uploaded
  • Information: Used Space / Total Available Space
  • A list of all files (with the file name and file extension) that the current user has
  • for each file we have 2 options: download and delete

My account

On My account page, we can set up 3 different cloud storage drives:

  • Google Drive
  • One Drive
  • Drop Box

For each one of them, we have a notification, informing us if we are looged or unlogged.

In case of an unlogged cloud account, we simply press on the button with the drive's name we want, and we will be redirected to the specific authentication page, then following some simple steps to sing in.

Log Out

If we want to disconnect from our account, we press the "Log out" button from navigation bar.
After disconnect is complet, you will be redirected to the main Informational page.
